# flow-docs 뷰어

이 폴더는 자기포함 기획/PRD 보고서 뷰어다. Python 3만 있으면 어디서나 실행된다(보기 전용).

## 실행

```bash
python3 serve.py prd.json
```

출력된 `http://localhost:8000`을 브라우저로 연다. 포트 변경: `--port 9000`.

## 보이는 것

- 보고서가 위에서 아래로 읽히고, 각 섹션 아래 상황에 맞는 차트가 인라인으로 렌더된다.
- 차트 타입: 순서도(흐름) / 수영선도 / 마인드맵 / 조직도 / 사이트맵 / 시퀀스 등.
- 잘못된 차트 1개는 그 자리에만 에러가 표시되고 나머지는 정상.

## 공유

이 `flow-docs/` 폴더를 그대로 커밋하면 팀원도 동일하게 실행할 수 있다(오프라인 동작).
