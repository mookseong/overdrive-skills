# flow-docs 뷰어

이 폴더는 자기포함 PRD 뷰어다. Python 3만 있으면 어디서나 실행된다.

## 실행

```bash
python3 serve.py prd.json
```

출력된 `http://localhost:8000`을 브라우저로 연다. 포트 변경: `--port 9000`.

## 사용

- **노드 선택**: 다이어그램 노드 또는 오른쪽 "노드 선택" 목록에서 클릭 → 그 단계 요구사항 표시.
- **편집**: "편집"에서 노드/엣지 추가·삭제·수정.
- **저장**: "저장" → `prd.json`에 기록된다.

## 공유

이 `flow-docs/` 폴더를 그대로 커밋하면 팀원도 동일하게 실행할 수 있다(오프라인 동작).
