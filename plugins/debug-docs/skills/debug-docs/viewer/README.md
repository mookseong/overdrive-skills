# debug-docs 뷰어

이 폴더는 자기포함 디버깅 다이어그램 뷰어다. Python 3만 있으면 어디서나 실행된다(보기 전용).

## 실행

```bash
python3 serve.py debug.json
```

출력된 `http://localhost:8000`을 브라우저로 연다. 포트 변경: `--port 9000`.

## 보이는 것

- 분석 요약(overview) + 상황에 맞는 다이어그램들이 순서대로 렌더된다.
- 다이어그램 타입: 시퀀스(호출 흐름) / 클래스(구조) / 상태(상태 전이) / 플로우차트(로직·원인 트리) / ER(데이터) 등.
- 잘못된 다이어그램 1개는 그 카드에만 에러가 표시되고 나머지는 정상.

## 공유

이 `debug-docs/` 폴더를 그대로 커밋하면 팀원도 동일하게 실행할 수 있다(오프라인 동작).
