# debug-docs 뷰어

이 폴더는 자기포함 RCA(원인 분석) 뷰어다. Python 3만 있으면 어디서나 실행된다.

## 실행

```bash
python3 serve.py debug.json
```

출력된 `http://localhost:8000`을 브라우저로 연다. 포트 변경: `--port 9000`.

## 사용

- **노드 선택**: 원인 트리 노드 또는 오른쪽 목록에서 클릭. 그 가설/원인의 증거가 표시된다.
- **상태 색**: 가설=파랑, 조사중=주황, 기각=회색, 확정(근본원인)=빨강.
- **편집**: 노드/연결 추가·삭제·수정, 노드 상태를 드롭다운으로 변경.
- **저장**: "저장" 누르면 `debug.json`에 기록된다.

## 공유

이 `debug-docs/` 폴더를 그대로 커밋하면 팀원도 동일하게 실행할 수 있다(오프라인 동작).
